・目標物：ロット（のプログラム）
ff14で誰がアイテムを入手するかを決める方法
ダイスの目が大きい人が手に入れることができる
1-99の範囲で目が出ます

・使い方
人数を数字で入力
プレイヤーの名前を入力(１名ずつ入力する）
人数分プレイヤーの名前が入力されると自動的にダイスを振ってアイテム入手者の判定を行います。


・今後の課題１
need,greed,pass(自分のジョブで使う、自分のジョブでは使えないが欲しい、いらない)の実装
needの1はgreedの99よりも強い。
passは0扱い。

・今後の課題２
同じ目は出ないようにする
ただしneed,greedが違う場合は同じ目も可能とする。（need99とgreed99は同時に存在可能）
